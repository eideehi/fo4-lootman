# Tools

This directory owns non-release maintenance and analysis tooling.

- `tools/native-hooks/` - native-hook address manifest, generator, verifier,
  resolver, and review bundles.
- `tools/ghidra/` - Ghidra headless config, scripts, projects, and reports.
- `tools/scripts/` - shared root preflight used by root `package.json`.
- `tools/tests/` - shared tool test helpers and tests for shared tooling.

Run commands from the repository root. See `docs/project-map.md`.
